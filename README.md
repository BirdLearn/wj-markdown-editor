## 特性

- 支持图片粘贴上传、截图上传等，细粒度控制图片保存方式
  ![](https://cdn.jsdelivr.net/gh/nlbwqmz/static-resource@main/image/540cf940dc4311ee9202979d26af7280.png)
- 图片预览支持放大、旋转等
- 主题切换
- 自动保存
- 自定义预览宽度
- 目录提取
- 导出为PDF
