## 特性

- **支持webdav同步**
- 支持本地MD文件打开
- 支持图片粘贴上传、截图上传等，细粒度控制图片保存方式
- 图片预览支持放大、旋转等
- 主题切换
- 自动保存
- 自定义预览宽度
- 目录提取
- 导出为PDF

## 注意
- **便携版**暂不支持自动升级，需手动下载，解压后直接替换根目录即可。
